# 中继点名活动管理系统操作文档

## 1. 系统简介

中继点名活动管理系统是一个专为业余无线电爱好者设计的活动记录管理平台，用于管理中继点名活动的基本信息、参与者记录以及数据导出功能。

### 主要功能
- 活动管理：创建、编辑、查看和删除中继点名活动
- 参与者记录：记录活动中的参与者信息，包括呼号、信号报告、设备、天线等
- 数据导出：支持将活动记录导出为Excel和ADIF格式
- 用户管理：管理系统用户，分配不同角色权限
- QTH地址管理：管理地理位置信息，支持模糊搜索

## 2. 功能模块说明

### 2.1 登录与注册

**登录**
- 输入用户名和密码进行登录
- 系统默认管理员账号：admin / password123

**注册**
- 普通用户可以注册账号
- 注册后默认为operator角色，只能参与活动记录录入

### 2.2 活动管理

**活动列表**
- 显示所有创建的活动，包括活动标题、外部ID、活动日期、呼号、频率等信息
- 显示活动的操作员呼号
- 支持活动的查看、开始活动、编辑和删除操作
- 支持下载活动列表为Excel文件

**创建/编辑活动**
- 活动标题：活动的名称
- 外部ID：活动的唯一标识
- 活动日期：活动举行的时间
- 呼号：活动使用的呼号
- 发射频率：活动使用的发射频率
- 接收频率：活动使用的接收频率（可选）
- 模式：活动使用的通信模式（SSB、CW、FM、AM）
- 波段：活动使用的波段（160m、80m、40m、20m、15m、10m、6m、2m、70cm）
- 操作员：选择参与活动的操作员，可以选择多个

### 2.3 活动详情与记录

**活动详情**
- 显示活动的基本信息
- 显示活动的参与者记录列表
- 支持查看参与者的详细信息
- 支持将活动记录导出为Excel和ADIF格式

**开始活动**
- 活动录入页面，用于记录参与者信息
- 输入参与者呼号，系统自动从历史记录中填充设备、天线、功率等信息
- 信号报告默认是59，可修改
- QTH输入框支持关键字搜索，匹配结构化地址
- 支持快捷键F1完成录入
- 参与者记录实时显示在下方，并支持搜索

### 2.4 数据导入导出

**活动记录导出**
- 选择活动，导出为Excel或ADIF格式
- Excel格式包含完整的活动和参与者信息
- ADIF格式符合标准，可用于其他系统导入

**所有活动记录导出**
- 一次导出系统中所有活动的完整记录
- 包含活动基本信息和参与者详情

**历史记录批量导入**
- 支持批量导入历史记录
- 提供导入模板下载

**QTH数据导入导出**
- 支持导入结构化的地理位置数据
- 支持导出QTH数据模板

## 3. 使用教程

### 3.1 登录系统

1. 打开浏览器，访问系统地址
2. 在登录页面输入用户名和密码
3. 点击"登录"按钮进入系统

### 3.2 创建活动

1. 登录系统后，点击左侧菜单中的"活动管理"
2. 点击"创建活动"按钮
3. 填写活动基本信息
4. 选择活动操作员
5. 点击"确定"按钮完成创建

### 3.3 开始活动记录

1. 在活动列表中，找到要记录的活动
2. 点击"开始活动"按钮
3. 在活动录入页面，输入参与者呼号
4. 系统自动填充历史记录中的设备、天线、功率等信息
5. 修改信号报告（如果需要）
6. 输入或选择QTH信息
7. 点击"录入"按钮或按F1键完成录入
8. 参与者记录将显示在下方列表中

### 3.4 导出活动记录

1. 在活动列表中，找到要导出的活动
2. 点击"查看"按钮进入活动详情页面
3. 在活动详情页面，点击"导出为Excel"或"导出为ADIF"按钮
4. 系统将生成并下载对应的文件

### 3.5 导出所有活动记录

1. 点击左侧菜单中的"导入导出"
2. 在"所有活动记录导出"卡片中，点击"导出所有活动记录"按钮
3. 系统将生成包含所有活动记录的Excel文件并下载

### 3.6 管理用户

1. 以管理员身份登录系统
2. 点击左侧菜单中的"用户管理"
3. 可以查看、添加、编辑和删除用户
4. 为每个用户分配唯一的呼号

### 3.7 导入QTH数据

1. 点击左侧菜单中的"导入导出"
2. 在"QTH数据导入导出"卡片中，点击"导出QTH数据"按钮下载模板
3. 填写QTH数据模板
4. 点击"点击或拖拽文件到此处上传"区域，选择填写好的模板文件
5. 点击"导入QTH数据"按钮完成导入

## 4. 常见操作流程

### 4.1 完整活动流程

1. 管理员创建活动，设置活动基本信息和操作员
2. 操作员登录系统，找到对应的活动
3. 点击"开始活动"，进入录入页面
4. 实时记录参与者信息
5. 活动结束后，导出活动记录
6. 管理员可以查看所有活动记录

### 4.2 数据备份与恢复

1. 定期导出所有活动记录作为备份
2. 如需恢复数据，可使用批量导入功能
3. 确保导入数据的格式符合模板要求

## 5. 注意事项

1. 每个用户只能有一个唯一的呼号
2. 活动外部ID必须唯一
3. 导出的ADIF文件符合标准格式，可直接用于其他系统导入
4. 录入参与者信息时，呼号输入后系统会自动填充历史数据
5. QTH输入支持关键字搜索，输入关键字后会显示匹配的结构化地址
6. 快捷键F1可快速完成录入

## 6. 故障排除

### 6.1 登录失败
- 检查用户名和密码是否正确
- 检查网络连接是否正常
- 联系管理员重置密码

### 6.2 活动创建失败
- 检查必填字段是否都已填写
- 检查活动外部ID是否已存在
- 检查操作员选择是否正确

### 6.3 导出功能异常
- 检查浏览器是否阻止了弹出窗口
- 检查网络连接是否正常
- 联系管理员检查系统日志

### 6.4 参与者记录保存失败
- 检查必填字段是否都已填写
- 检查网络连接是否正常
- 刷新页面后重试

## 7. 联系方式

如有任何问题或建议，请联系系统管理员。
